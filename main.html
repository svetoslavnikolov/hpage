Showing your publications


The easiest way to show your publications is to use the following file: publications.php


This is how you do it:

  1. Download the file publications.php. Remember to set the extension ".php", not ".txt".
  2. Add a link in the menu to this file: 
    add_menu('left', 'Publications', 'directory/', 'publications.php');
  3. Finally, replace the first argument in the call to pubdb_page() from 12 to the your own ID number.


Welcome to the small web site of H-PaGe, which is short from Home Page Generator. HPaGe was designed with only one thoght in mind - create web pages like this as easily as possible. The whole layout of the page is created by a single script "index.php". The only thing that the user must do is to provide a single file, called "main.html" which is the actual contents of the page.

The menu on the left side of the page can be created either from a specification given in a file, or from the directory structure. You can get a presentation of H-PaGe:
